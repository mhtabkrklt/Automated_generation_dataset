# Доверительные интервалы результатов оценки моделей

Метод: bootstrap (10 000 итераций), уровень доверия 95%.  
Метрика: Final Accuracy — точное совпадение с эталоном даёт 1.0, частичное (по оценке LLM-судьи) — 0.5, неверный ответ — 0.0.  
Выборка: 309 задач на каждую модель (диалоговый + графовый пайплайны, три уровня сложности).

---

## Общий результат

| Модель | Accuracy | 95% CI |
|---|---|---|
| glm-5p1 | 0.715 | [0.663, 0.764] |
| kimi-k2p6 | 0.761 | [0.712, 0.809] |
| gpt-5-mini | 0.725 | [0.675, 0.773] |

Kimi-k2p6 показывает лучший результат, однако интервалы всех трёх моделей перекрываются — статистически значимых различий между моделями при данном объёме выборки нет.

По knowledge_update все три модели дали одинаковый результат (0.672) — это наиболее трудный тип для всех участников. По composite зафиксирована наибольшая разница: kimi-k2p6 (0.781) опережает glm-5p1 (0.625), их интервалы почти не пересекаются.

## Выводы

1. Уровень сложности — единственный фактор, где различия статистически значимы для всех моделей и всех типов задач.
2. Графовый пайплайн лучше разделяет модели, чем диалоговый.
3. Единственная достоверная разница между моделями — composite: kimi-k2p6 превосходит glm-5p1.
4. Для получения статистически значимых различий между моделями в общем зачёте необходимо увеличить выборку примерно до 600–700 задач на модель.
